## 🗄️ Database Structure (MySQL)

### Tables Required

1. **users** - User accounts
2. **admins** - Admin accounts
3. **gestures** - Gesture library (predefined signs)
4. **gesture_history** - User gesture detection logs
5. **language_preferences** - User language settings
6. **training_datasets** - Uploaded training data

## 🛠️ Installation Steps

### Step 1: Prerequisites
```bash
# Install Python 3.8+
# Install WAMP Server
# Start WAMP Server (Apache + MySQL)
```

### Step 2: Create Virtual Environment
```bash
cd SignAI
python -m venv venv
venv\Scripts\activate  # Windows
source venv/bin/activate  # Mac/Linux
```

### Step 3: Install Dependencies
```bash
pip install flask
pip install mysql-connector-python
pip install opencv-python
pip install mediapipe
pip install tensorflow
pip install numpy pandas
pip install googletrans==4.0.0-rc1
pip install gTTS
pip install Pillow
```

### Step 4: Setup Database
```bash
python create_db.py
```

### Step 5: Run Application
```bash
python app.py
```

Visit: `http://localhost:5000`

## 🔧 Configuration Setup

### config.py
```python
# Database
DB_HOST = 'localhost'
DB_USER = 'root'
DB_PASSWORD = ''
DB_NAME = 'signai_db'

# Flask
SECRET_KEY = 'your-secret-key-here'
DEBUG = True

# Upload settings
UPLOAD_FOLDER = 'uploads/'
MAX_CONTENT_LENGTH = 16 * 1024 * 1024  # 16MB

# AI Model
MODEL_PATH = 'static/models/gesture_model.h5'
CONFIDENCE_THRESHOLD = 0.75

# Emergency
EMERGENCY_GESTURES = ['HELP', 'SOS', 'DANGER']
ADMIN_EMAIL = 'admin@signai.com'

# Translation
LANGUAGES = {
    'tamil': 'ta',
    'hindi': 'hi'
}
```

## 🧠 AI Model Training Flow

### Data Collection
1. Admin uploads gesture videos/images
2. System extracts hand landmarks
3. Data is labeled by admin
4. Dataset stored in database

### Model Training
1. Load labeled dataset
2. Preprocess hand landmarks
3. Train neural network (CNN/LSTM)
4. Validate accuracy
5. Save best model (.h5 file)
6. Deploy to production

### Inference (Real-time)
1. Capture video frame
2. Detect hand landmarks
3. Extract features
4. Predict gesture class
5. Return result with confidence score
